
## Environment Setup

1. **Clone the Repository**

   ```bash
   git clone https://github.com/BlockNKey/rattad-next.git
   cd rattad-next
   ```

2. **Install Dependencies**

   ```bash
   npm install
   ```

3. **Run Development Server**
   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   # or
   bun dev
   ```
